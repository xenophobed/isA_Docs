# isA Vibe

Your AI coding assistant, like Claude Code.

## Overview

isA Vibe is a powerful coding assistant built on the isA Agent SDK. It helps developers write, debug, and refactor code with natural language interactions.

## Features

### Code Understanding
- Read and analyze codebases
- Explain complex code
- Find bugs and issues
- Suggest improvements

### Code Generation
- Write new code from descriptions
- Generate tests
- Create documentation
- Implement features

### Refactoring
- Improve code structure
- Apply best practices
- Modernize legacy code
- Optimize performance

### Multi-Language Support
- Python, JavaScript, TypeScript
- Go, Rust, Java
- C, C++, C#
- And many more

## Quick Start

```bash
# Install isA Vibe CLI
pip install isa-vibe

# Start coding
isa-vibe "Help me refactor this function"
```

## Skills

isA Vibe comes with specialized skills:
- `code-review` - Expert code review
- `debug` - Systematic debugging
- `refactor` - Code improvement
- `test-writer` - Test generation
- `documentation` - Doc generation

## Configuration

```yaml
# .isa-vibe.yaml
skills:
  - code-review
  - debug
allowed_tools:
  - read_file
  - write_file
  - bash
max_iterations: 50
```
