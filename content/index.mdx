## Quick Start

### Install the SDK

```bash
pip install isa-agent-sdk
```

### Create Your First Agent

```python
from isa_agent_sdk import Agent, query

# Simple query
async for msg in query("What is the weather in Tokyo?"):
    print(msg.content, end="")

# Custom agent with tools
agent = Agent(
    name="my-assistant",
    model="claude-sonnet-4-20250514",
    tools=["web_search", "calculator"]
)

response = await agent.run("Search for recent AI news")
```

### Deploy to Production

```bash
# Deploy with isA Cloud
isa deploy --env production
```
