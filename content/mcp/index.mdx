# ISA MCP Platform

Enterprise-grade Model Context Protocol (MCP) platform for tool registration, discovery, invocation, and external MCP aggregation.

## Overview

ISA MCP provides:

- Unified tool/prompt/resource registry
- Skill-based semantic discovery
- Human-in-the-loop (HIL) workflows
- External MCP server aggregation
- Multi-tenant isolation and security controls

## Latest Updates (2026-02-14)

- **Hybrid multi-tenant platform rollout** across tool, prompt, resource, skill, marketplace, and aggregator services.
- **Security hardening** for auth bypass prevention, shell execution safeguards, path traversal protection, and registry fetch SSRF controls.
- **Reliability improvements** for Qdrant retry handling, vector ID overflow protection, cache versioning, and atomic delete flows.
- **Guidance refresh** with new security, multi-tenant, and reliability operational docs.

## Core Features

### Skill-Based Semantic Search

Natural-language query routing through skill classification and ranked matches.

```python
results = await search("schedule a meeting with John")
```

### Human-in-the-Loop

Supports:

- authorization
- input collection
- review
- combined input + authorization

### Aggregator Service

Register, connect, monitor, and route to external MCP servers with namespace-aware tool routing.

## API Endpoints

Base URL: `http://localhost:8081`

| Endpoint | Method | Description |
|----------|--------|-------------|
| `/mcp` | POST | MCP JSON-RPC endpoint |
| `/health` | GET | Server health check |
| `/search` | POST | Semantic search |
| `/progress/{id}/stream` | GET | SSE progress stream |
| `/api/v1/skills/*` | - | Skill management |
| `/api/v1/aggregator/*` | - | External server aggregation |
